Quick API Documentation

POST /api/v1/profiles

Description: Create profile from LinkedIn URL

curl -X POST http://localhost:5000/api/v1/profiles \ -H "X-API-Key: YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "linkedinURL": "https://linkedin.com/in/example", "name": "John Doe" }'

POST /api/v1/matches

Description: Find collaboration matches

curl -X POST http://localhost:5000/api/v1/matches \ -H "X-API-Key: YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "profileId": "PROFILE_ID", "limit": 10, "minScore": 50 }'

GET /api/v1/profiles/:id

Description: Get specific profile

curl -X GET http://localhost:5000/api/v1/profiles/PROFILE_ID \ -H "X-API-Key: YOUR_API_KEY"
